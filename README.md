# 🌼 Flores Amarillas

A small interactive web experience built to tell a story in a soft, aesthetic, and emotional way — inspired by the idea of "flores amarillas".

This project focuses on smooth animations, storytelling through scroll, and a clean, modern UI.

---

## ✨ About the Project

This is a front-end storytelling project where the user scrolls through different moments of a story.
Each section reveals new elements like chat bubbles, characters, and animations, creating a dynamic and emotional experience.

The goal was not just to build something functional, but something that *feels* good.

---

## ⚡ Built With

* React (Vite)
* TailwindCSS
* Framer Motion

---

## 🎯 Features

* Scroll-based storytelling
* Animated chat bubbles
* Character-based scenes
* Floating petals animation 🌸
* Smooth transitions between story steps
* Final reveal interaction

---

## 🧠 Process

This project was heavily **vibe coded**.

Instead of following a rigid plan, it was built iteratively — testing ideas, adjusting visuals, and improving interactions step by step.

A lot of the structure, animations, and creative direction were explored using AI assistance, combined with my own adjustments and refinements.
